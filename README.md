AdventureWorks Power BI Dashboard

This repository contains a full end-to-end Power BI project built as part of a Udemy Power BI course.
The project uses the AdventureWorks dataset and demonstrates the complete BI workflow ‚Äî including data preparation, modelling, DAX calculations, and dashboard design with insights into revenue, profit, orders, returns, customer behaviour, and product performance.

 Project Overview

This Power BI solution showcases:

Data cleaning & transformation using Power Query

Star-schema modelling with relationships between fact and dimension tables

Custom DAX measures for KPIs such as Revenue, Profit, Orders, Return Rate, YoY growth

Interactive visualizations including line charts, category breakdowns, maps, KPIs, slicers & tables

Business insights around trends, top-selling products, returns, and customer behaviour

The dashboards are designed to simulate real-world BI reporting used by business leaders to make data-driven decisions.
